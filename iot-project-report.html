Aperçu du projet

Ce projet réalise une serre agricole intelligente divisée en 4 zones indépendantes. Chaque zone est équipée de capteurs connectés à un Arduino qui transmettent leurs mesures via port série à Node-RED. Le système compare les données en temps réel aux valeurs idéales stockées dans InfluxDB pour la plante sélectionnée, puis déclenche automatiquement les actionneurs (thermostat, pompe, éclairage).

Une intégration avec l'API OpenWeatherMap surveille les conditions météorologiques locales (Tizi-Ouzou, Algérie) et contrôle automatiquement l'ouverture du dôme de la serre : il s'ouvre par temps dégagé ou peu nuageux, et se ferme dans tous les autres cas.

Dans chaque zone, l'utilisateur sélectionne la plante cultivée via un menu déroulant sur le dashboard. Les seuils idéaux (température, humidité, luminosité, insectes nuisibles) sont récupérés depuis InfluxDB pour cette plante, ce qui permet d'adapter les actionneurs à chaque culture.

Insectes nuisibles détectés

  • Puceron — insecte suceur de sève
  • Punaise — ravageur des cultures
  • Limace — gastéropode dévorant les feuilles

En cas de détection, une notification est envoyée et le nom de l'insecte est affiché sur le dashboard.

Organisation des données InfluxDB

Bucket : final

  • Measurement : maSerre
  • Fields : temp, humidity, luminosity
  • Fields : insectOne, insectTwo
  • Tag : zone (zone1 → zone4)
  • Valeurs idéales par type de plante

Bucket : choix

  • Measurement : choix_zone1 … choix_zone4
  • Field : value (nom de la plante sélectionnée)
  • Persistance du choix de plante entre les sessions

Installation et Configuration

Les clés API et tokens ne sont pas inclus dans ce dépôt. Vous devez configurer vos propres credentials après import des flows.
1

Installer Node-RED et les palettes

Node.js requis. Installer les palettes depuis le gestionnaire de Node-RED :

node-red-dashboard
node-red-node-openweathermap
node-red-contrib-influxdb
2

Importer les flows

Dans Node-RED : Menu (☰) → Import → coller le contenu de greenhouse-main-flows.json

Pour la zone 1 modifiée, importer aussi greenhouse-zone1-flows.json

3

Configurer InfluxDB

Double-cliquer sur n'importe quel noeud InfluxDB → modifier la configuration Smart Green House :

URL    : https://votre-instance.influxdata.com
Token  : YOUR_INFLUXDB_TOKEN
Org    : YOUR_INFLUXDB_ORG
Bucket : final   (données capteurs)
Bucket : choix   (sélection de plantes)
4

Pré-remplir les valeurs idéales dans InfluxDB

Écrire dans le bucket final, measurement maSerre, les valeurs idéales pour chaque plante et chaque zone (température, humidité, luminosité, insectes nuisibles).

5

Configurer OpenWeatherMap

Double-cliquer sur le noeud weather → entrer votre clé API et la ville souhaitée.

6

Connecter l'Arduino

Configurer les noeuds serial in avec le bon port COM. L'Arduino envoie les données au format topic:valeur\n (ex : temperature:23\n, soilHumidity:65\n).
